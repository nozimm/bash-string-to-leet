# String to Leet
Convert any String to Leet with Bash

## Installation

```
git clone https://github.com/nozimm/bash-string-to-leet && cd bash-string-to-leet
sudo chmod a+rx leet && sudo mv leet /usr/local/bin/leet
cd .. && rm -rf ash-string-to-leet
```

## Usage
```
leet "YOUR STRING"
```
Output: ```Y0UR 57R1NG```
